# Information sheet about making discord servers
## Basic Information
Okay, in this sheet you will learn how to make great Discord servers, so we have a few types of Discord servers that exist.
We will handle in this document both vanilla and default. 
You should design a server to make sure it fits all the needs, has sensible categories, and also isn't too bloated and only has necessary stuff. Some users will like really searching with a lot of channels for every purpose, and some users will like to have a lot of purposes combined into one channel. This is just a person's preference. Also, the style of split signals is a preference, for example clean and minimal versus detailed unicodes. Also, the emojis that are used will also differ per server. Some people like these office corporatey chill emojis, and some people like these extravagant ones. This is all up to the user, but you should take it into account when designing.  It is often a bad idea to have the same emoji in multiple things, but if you have, for example, two log channels, then it is acceptable to use the same emoji. 
Always capitalize the first letter of a category, and you don't have to put the split symbols in category names. 

These are all the split signals you will get to use. Never use space, as that is ugly unless it is allowed by the style.

## Server Styles
### Vanialla
1. Vanilla: this means you will not use any emojis in the channel names, and you will also not use any special split signals. It will just be spaces, which Discord will convert to dashes. In the category names, you will do an emoji with a separator.
These servers are often quite minimal, but if they are big, they will become ugly. 

### Default
In the default style, what you will do is make category names probably without emoji, and then just make the categories. This will have an emoji, then a splitter, then the name. In the name, if it is community board, you can do a space, which would become a dash, but only for those longer names. I don't want, for example, discord announcements to be one thing.

You should really use emojis that correspond to the channel, and if it's more of a community one, make a nice community for maybe a flower or a nice symbol. You should also use some splitters, which I included, which are two and can be used to fence off a word, keep in mind and so on, so are just one separator wall.
